# Personal Portfolio Website

This repository contains the source code for my personal developer portfolio, built with HTML, Tailwind CSS, and JavaScript. It showcases my software engineering projects, experience, and background, and is deployed using GitHub Pages.

## Live Site
https://adogins.github.io/resume-site/

## Features
- Modern, responsive design
- Light/Dark mode toggle with local storage
- Lavender-themed UI with custom Tailwind palette
- Medium glow effect on GitHub buttons in dark mode
- Unified Projects section with detailed cards
- Secure email injection using GitHub Secrets (no email stored in repo)

## Tech Stack
- **HTML5**
- **Tailwind CSS**
- **JavaScript**
- **GitHub Pages**
- **GitHub Actions** (for deployment + email injection)
